admin.v2alpha.Certificates
--------------------------

`[admin.v2alpha.Certificates proto] <https://github.com/envoyproxy/envoy/blob/5c801b25cae04f06bf48248c90e87d623d7a6283/api/envoy/admin/v2alpha/certs.proto#L18>`_

Proto representation of certificate details. Admin endpoint uses this wrapper for `/certs` to
display certificate information. See :ref:`/certs <operations_admin_interface_certs>` for more
information.

.. code-block:: json

  {
    "certificates": []
  }

certificates
  (**repeated** :ref:`admin.v2alpha.Certificate <envoy_api_msg_admin.v2alpha.Certificate>`) List of certificates known to an Envoy.
