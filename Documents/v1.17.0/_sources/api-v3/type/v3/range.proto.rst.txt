type.v3.Int64Range
------------------

`[type.v3.Int64Range proto] <https://github.com/envoyproxy/envoy/blob/5c801b25cae04f06bf48248c90e87d623d7a6283/api/envoy/type/v3/range.proto#L16>`_

Specifies the int64 start and end of the range using half-open interval semantics [start,
end).

.. code-block:: json

  {
    "start": "...",
    "end": "..."
  }
