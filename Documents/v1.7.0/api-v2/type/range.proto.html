Range¶

type.Int64Range¶

[type.Int64Range proto]

Specifies the int64 start and end of the range using half-open interval semantics [start, end).

{
  "start": "...",
  "end": "..."
}
start
(int64) start of the range (inclusive)
end
(int64) end of the range (exclusive)
